MOBILYZER

THE

CHALLENGE

Our actual task was to create a tool which with the help of google will check the website's mobile based performance i.e how mobile friendly is the website or how fast does it opens in mobile browsers. Based on those results the mobile performance of the website can be improved to a great extent.

OUR

APPROACH

To complete this task first we study about what are the features that affects the mobile performance of the website. Then on the basis of these criteria and with the help of Google we created a tool which tells the mobile performance of the website and generates some feedback on how to improve the mobile performance and speed of the website.

We aimed at creating a tool which tells website's mobile performnace and helps us to increase it and to know how much our website is mobile friendly and mobile fast.
